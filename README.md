# Basic Python Programs

This repository contains beginner-level Python programs
written while learning Python.

## Programs Included
- Add two numbers
- Find maximum of two numbers
- Factorial of a number
- Simple and compound interest calculation
- Electricity bill
- Data bill
- Armstrong Number
- Count Number
- Check even or odd
- Largest and Smallest digits
- Palindrome
- Product of digits
- Remove last digit of number
- Reversed Number
- Sum of digits
